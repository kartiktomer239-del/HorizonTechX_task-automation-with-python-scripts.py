📌 Overview
This project demonstrates basic task automation using Python scripts. It covers small, real‑life repetitive tasks like moving files, extracting emails, or scraping webpage titles.

⚙️ Features
Move .jpg files → Transfer all .jpg images from one folder to another.

Extract emails → Find email addresses in a .txt file and save them to a new file.

Scrape webpage title → Fetch the <title> of a webpage and store it locally.

🛠️ Key Concepts
os → File and folder operations

shutil → Moving files

re → Regular expressions

requests → Web scraping

File handling → Reading and writing files

🚀 How to Run
Save the script as automation.py.

Run in terminal:


python automation.py
Follow the menu prompts to perform your chosen task.
